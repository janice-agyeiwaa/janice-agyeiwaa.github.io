# janice-agyeiwaa.github.io

Personal portfolio website for **Janice Agyeiwaa Owusu** — Geomatics professional and GIS specialist based in Vancouver, BC.

🌐 **Live site:** [https://janice-agyeiwaa.github.io](https://janice-agyeiwaa.github.io)

---

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Animations:** Framer Motion
- **Deployment:** GitHub Pages
- **Language:** TypeScript

---

## Running Locally

### Prerequisites
- Node.js 18+
- npm or yarn

### Steps

1. Clone the repository:
```bash
git clone https://github.com/janice-agyeiwaa/janice-agyeiwaa.github.io
cd janice-agyeiwaa.github.io
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Updating Content

All personal information lives in one place — `data/profile.json`. Edit that file to update your bio, skills, experience, projects, education, and certifications. The site updates automatically.

### Building for Production

```bash
npm run build
```
